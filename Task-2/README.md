# 🛒 Quantium Retail Analytics – Trial Store Analysis

## 📌 Project Overview

This project analyzes the impact of a retail store trial conducted as part of the **Quantium Data Analytics Virtual Experience**.

The objective is to identify suitable control stores, compare their performance with trial stores, and evaluate whether the new store layout resulted in a statistically significant improvement in sales.

---

## 🎯 Business Objective

Evaluate the performance of the following trial stores:

- Store 77
- Store 86
- Store 88

The analysis aims to:

- Select the most appropriate control store.
- Compare monthly KPIs before and during the trial.
- Scale control store sales for fair comparison.
- Measure sales uplift during the trial period.
- Assess statistical significance using a **95% confidence interval**.
- Provide business recommendations on whether the new layout should be rolled out.

---

## 🛠️ Technologies Used

- Python

- Matplotlib

---

## 📊 Analysis Workflow

1. Create monthly store KPIs
2. Define the pre-trial period
3. Calculate Pearson Correlation
4. Calculate Magnitude Similarity
5. Select the best control store
6. Compare trial and control store performance
7. Scale control store sales
8. Calculate sales uplift
9. Assess statistical significance using a 95% confidence interval
10. Generate business insights and recommendations

---

## 📈 Key Visualizations

- Similarity Score Comparison
- Monthly Sales Comparison
- Monthly Customer Comparison
- Average Transactions per Customer
- Scaled Sales Comparison
- Confidence Interval Analysis
- Trial Performance Summary

---

## 📋 Results

The analysis identified the most suitable control store for each trial store and evaluated the effectiveness of the trial layout using statistical analysis.

The final recommendation indicates whether the new layout should be rolled out across similar stores.
